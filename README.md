# Tasty Bites - Full Stack Restaurant Website

A professional, real-world MERN (MongoDB, Express, React, Node) restaurant website designed for local businesses. This project also features a dedicated "Freelancer Showcase" section, making it an ideal portfolio piece for developers.

## 🚀 Features

- **Dynamic Menu**: Categorized menu with search and filter functionality.
- **Online Reservations**: Simple and effective booking form with instant confirmation.
- **Admin Dashboard**: Manage your menu items (CRUD) and view all incoming reservations.
- **Contact System**: Direct messaging for general enquiries.
- **Freelancer Showcase**: A dedicated section to highlight the developer's skills and attract clients.
- **Responsive Design**: Mobile-first approach built with Tailwind CSS v4.
- **Smooth Animations**: Powered by Framer Motion.

## 🛠️ Tech Stack

- **Frontend**: React.js, Tailwind CSS 4, Axios, React Router, Lucide React, Framer Motion.
- **Backend**: Node.js, Express.js.
- **Database**: MongoDB (Mongoose).

## 📂 Project Structure

- `/client`: React frontend (Vite).
- `/server`: Node.js/Express backend.

## 🚦 Getting Started

### 1. Prerequisites
- Node.js & npm installed.
- A MongoDB database (local or Atlas).

### 2. Backend Setup
1. Navigate to the `server` directory: `cd server`
2. Create a `.env` file based on `.env.example`:
   ```env
   PORT=5000
   MONGO_URI=your_mongodb_connection_string
   ```
3. Install dependencies: `npm install`
4. Seed the database with sample items: `npm run seed`
5. Start the server: `npm run dev`

### 3. Frontend Setup
1. Navigate to the `client` directory: `cd client`
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
